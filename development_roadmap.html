const res = await fetch('/api/crawlers/'); const {results} = await res.json();
Step 2
Wire Run Crawler button
Run button calls POST /api/crawlers/{id}/run/. Django triggers ECS Fargate task via boto3. Button shows spinner and live ECS status.
Step 3
Wire Heal button
Heal button calls POST /api/crawlers/{id}/heal/. Django creates HealAttempt, calls MCP server. Dashboard polls /api/heal-log/{id}/ every 5s for progress.
Step 4
Date range filters
Filter selectors send date_from + date_to query params to /api/crawlers/. Django filters SpiderRuns by date range. Results update instantly.
Step 5
Auto-refresh every 60s
setInterval polling re-fetches spider statuses every 60 seconds. Status dots update live. Running spiders show animated pulse.
setInterval(loadCrawlers, 60000);

Deploy frontend via ECS
Copy dashboard HTML into dashboard/templates/dashboard.html. Django serves it at /. ALB routes port 443 to ECS service. Done.

Phase 3 — Claude agentic loop
Implementation steps
Step 1
Build FastAPI MCP server skeleton
Create mcp_server/server.py. FastAPI app with /heal endpoint, /health check, Claude agentic loop with 15-minute wall-clock timeout.
POST /heal → start Claude loop GET /health → {"status":"ok"}
Step 2
Build spider_tools.py (4 tools)
classify_failure() — reads error log, returns failure type. test_url_strategy() — tests one bypass. try_all_strategies() — tries all 4. fetch_page_html() — downloads live HTML.
Step 3
Build github_tools.py (2 tools)
get_spider_code() — fetches spider Python source via GitHub Contents API. commit_spider_fix() — pushes complete fixed file to ford-lens-auto-heal branch only.
Step 4
Build ecs_tools.py (2 tools)
trigger_ecs_run() — starts Fargate task with CMD override for spider name. poll_task_status() — polls DescribeTasks every 30s until STOPPED.
Step 5
Create ford-lens-auto-heal branch
Create the branch. Add GitHub branch protection rules restricting Claude's token to car_forums/spiders/** only. Cannot touch pipelines, settings, workflows.
git checkout -b ford-lens-auto-heal # Branch protection: spiders/** only
Step 6
Write CLAUDE.md
Codebase conventions: middleware classes, XPath patterns, item fields, how spiders are structured, how to add null guards, commit message format.
CLAUDE.md → always read by Claude # Spider conventions # Middleware classes # XPath patterns
Step 7
Wire run_summary_crawler tool
Claude calls this tool before committing. Runs the spider's companion test script against 1–2 pages. Returns passed:bool + which of the 7 checks failed.
Step 8
Deploy MCP server to ECS
Deploy ford-lens-mcp ECS service (always-on). Set stopTimeout: 900s (15 min). Add ANTHROPIC_API_KEY and GITHUB_TOKEN to task definition env vars.
